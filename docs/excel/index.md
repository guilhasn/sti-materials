# Modulo Excel para Decisao Publica

Este modulo cobre as **3 primeiras aulas** da UC, com foco em analise operacional na Administracao Publica.

## Pre-requisitos

- Excel 2019+ ou Microsoft 365
- Conhecimentos basicos de folhas de calculo (navegacao, filtros, ordenacao)
- Leitura de tabelas de dados com cabecalhos

## Software recomendado

- Microsoft Excel (desktop)
- Visualizador de PDF para fichas
- GitHub para descarregar e submeter versoes de trabalho

## Calendario sugerido (3 semanas)

- Semana 1: Aula 1 - Tabelas Dinamicas e agregacoes
- Semana 2: Aula 2 - Funcoes condicionais basicas
- Semana 3: Aula 3 - Caso integrador e recomendacao de gestao

## Entregaveis por aula

- `dataset.xlsx`
- `ficha.pdf`
- `resolucao_docente.xlsx`
