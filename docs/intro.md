# Sistemas e Tecnologias de Informacao (MAP)

Este site agrega os materiais da UC de **Sistemas e Tecnologias de Informacao** para o bloco inicial de **Excel para decisao publica**.

## Estrutura do modulo

- Aula 1: Tabelas Dinamicas + agregacoes (`CONTAR`, `SOMA`, `MEDIA`, `MAXIMO`, `MINIMO`)
- Aula 2: Funcoes condicionais basicas (`CONTAR.SE`, `SOMA.SE`, `SE`)
- Aula 3: Caso integrador com analise completa para contexto da Administracao Publica

## Publico-alvo

Estudantes do Mestrado em Administracao Publica (ESTG Leiria), com foco em decisao baseada em dados.

## Como trabalhar

1. Descarregar o `dataset.xlsx` de cada aula.
2. Resolver a `ficha.pdf`.
3. Confirmar o resultado com `resolucao_docente.xlsx`.
