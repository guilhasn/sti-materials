## Como trabalhar

1. Aceder a pagina da aula correspondente.
2. Descarregar o dataset e a ficha de exercicios.
3. Resolver os exercicios com o software indicado.
4. Confirmar resultados com a resolucao do docente (quando disponivel).

!!! info "Caso ancora"
    Todas as aulas utilizam o cenario ficticio mas realista da **Camara Municipal de Pombal** a gerir as consequencias da **Tempestade Kristin**. Este fio condutor liga Excel, BPMN, modelacao de dados e Power BI num contexto unico de administracao publica.
